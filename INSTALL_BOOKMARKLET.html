How to Use

  1. Go to a LinkedIn profile page
    URL should look like: linkedin.com/in/someone
  2. Click your bookmarklet (or paste code in console)
    A panel appears in the top-right
  3. Click "Extract This Profile"
    Data is saved to your browser
  4. Visit more profiles and repeat
    Counter shows how many you've saved
  5. Click "Export to CSV" when ready
    Downloads all saved profiles

Data Captured

  • Name - Full name
  • Headline - Professional tagline
  • Location - City/Region
  • Current Title - Job title
  • Current Company - Employer
  • Profile URL - Direct link
  • Date - When extracted
Tip: Scroll down on the profile before extracting to make sure all sections are loaded.
